# pyharness website

Marketing and docs site for pyharness, built with [TanStack Start](https://tanstack.com/start), Vite, Tailwind v4, and shadcn/ui.

## Getting Started

Install dependencies:

```bash
bun install
```

Run the dev server:

```bash
bun run dev
```

## Building for Production

```bash
bun run build
```

Preview the production build locally:

```bash
bun run preview
```

## Assets

Placeholder assets and TODOs are tracked in [`public/ASSETS_TODO.md`](./public/ASSETS_TODO.md).

## Deployment

The site deploys to [Vercel](https://vercel.com/). Nitro's Vercel preset is configured in `vite.config.ts`.
